Installation

Get started with Forge UI by installing the package via npm or yarn. Follow the steps below to add Forge UI to your project.

Requirements

Before installing Forge UI, make sure you have the following prerequisites:

  • Node.js 16.0 or later
  • React 18.0 or later
  • npm 7+ or yarn 1.22+

Package Installation

Install Forge UI using your preferred package manager:

npm 
npm install @forge-ui/react
yarn 
yarn add @forge-ui/react
pnpm 
pnpm add @forge-ui/react

Provider Setup

Wrap your application with the ForgeProvider component to enable theming and global configuration:

jsx 
import { ForgeProvider } from '@forge-ui/react';

function App() {
  return (
    <ForgeProvider theme="dark">
      <YourApp />
    </ForgeProvider>
  );
}

export default App;

Tip

The ForgeProvider should be placed at the root of your application to ensure all components have access to the theme context.

Basic Usage

Once installed, you can import and use any component from Forge UI:

jsx 
import { Button, Card, Input } from '@forge-ui/react';

function LoginForm() {
  return (
    <Card padding="lg">
      <Input
        label="Email"
        type="email"
        placeholder="you@example.com"
      />
      <Input
        label="Password"
        type="password"
      />
      <Button variant="primary" fullWidth>
        Sign In
      </Button>
    </Card>
  );
}

ForgeProvider Props

The following props are available for configuring the provider:

Prop Type Default Description
theme 'light' | 'dark' 'light' The color theme for all components
childrenRequired ReactNode — Your application components
colorScheme object defaultColors Custom color overrides
disableAnimations boolean false Disable all CSS animations

TypeScript Support

Forge UI is written in TypeScript and includes type definitions out of the box. No additional packages are required.

TypeScript Version

Forge UI requires TypeScript 4.7 or later for full type inference support.
